# Expense Tracker 💰

This is a full-stack web project to manage income and expenses.

## Features
- User Login & Register
- Add Income & Expense
- View Total Balance
- Transaction History

## Tech Used
- HTML, CSS, Bootstrap
- PHP
- MySQL
